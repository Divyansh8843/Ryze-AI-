## üì¶ Installation & Setup

Get running in **3 Minutes**.

### 1. Clone the Repo
```bash
git clone https://github.com/Divyansh8843/Ryze-AI-.git
cd Ryze-AI-
```

### 2. Ignite the AI Brain üß†
```bash
cd ai-service
# Install dependencies

# Start the Neural Engine
python app.py
```
